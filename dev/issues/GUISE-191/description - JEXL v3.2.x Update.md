# `GUISE-191`: Update JEXL to v3.2.x.

It looks like JEXL 3.2.1 is currently available (see [release notes](https://commons.apache.org/proper/commons-jexl/changes-report.html#a3.2.1)), but since it was stuck at v3.1 for so long, it's best not to upgrade while Guide 5.x is going out the door. Update JEXL in Guise 6.x, and see if some of the tickets it fixes are helpful (e.g. [JEXL-340](https://issues.apache.org/jira/browse/JEXL-340) for providing better error messages).
